Flash the UberSDR CYD display firmware directly from your browser using ESP Web Tools. No drivers, no Python, no PlatformIO required — just Chrome or Edge and a USB cable.

WiFi & UberSDR server
Fill these in and they are baked into the flash image — the display boots straight onto your network and starts showing your UberSDR instance. Leave blank to configure later on-screen or via the setup hotspot.

⚠️ Web Serial is not available in this browser.
Please open this page in Google Chrome or Microsoft Edge (desktop, version 89+). Firefox and Safari do not support Web Serial.
⚠️ Web Serial requires a secure context (HTTPS).
This page must be served over https:// — opening the file directly from disk (file://) will not work.
  1. Plug the CYD into this computer with a USB data cable (micro-USB port on the board).
  2. Fill in your settings above (or leave blank).
  3. Click Install, pick the serial port that appears, and follow the prompts.
  4. When asked, choosing Erase device is recommended for a first install.
⚠️ If no serial port appears, your cable may be charge-only, or Windows may need the CH340 driver (the CYD's USB-serial chip). Linux and macOS need no driver.
⚠️ "Failed to initialize"? The browser couldn't switch the board into download mode — common with the CYD's CH340 chip. First close anything else using the serial port (e.g. a serial monitor). Then hold the BOOT button (next to RST on the board edge) while clicking Install, keeping it held until writing starts. Still stuck? With BOOT held, tap RST, then release BOOT a second later — the board is now in download mode — and click Install again. Tap RST after flashing if it doesn't reboot by itself.

After flashing

  1. The device reboots. With settings baked in, it joins your WiFi and starts the overview slideshow — done.
  2. Without baked-in WiFi, connect to the UberSDR-Setup hotspot (password ubersdr1) and browse to http://192.168.4.1/, or just tap the screen to use the on-screen keyboard.
  3. Anything can be changed later at http://ubersdr-cyd.local/ — WiFi, server details, brightness.
